# StudioFlow - Sistema de Gestão de Agendamentos

StudioFlow é um sistema front-end completo para gerenciamento de agendamentos em pequenas empresas, estúdios e prestadores de serviço. Desenvolvido com foco em design profissional, usabilidade e estrutura escalável para implementação futura de back-end.

## 🔍 Funcionalidades

- 📊 **Dashboard** com resumo de agendamentos, faturamento e clientes
- 📅 **Tabela interativa** de agendamentos recentes (simulada em JS)
- 📁 Estrutura modular com navegação entre seções
- 🎨 Design responsivo com layout moderno
- 🖼️ Imagem de logo incluída
- ⚙️ Pronto para integração com back-end ou SPA

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (DOM Manipulation)
- Design responsivo (Grid Layout)
- PNG para branding visual

## 📂 Estrutura do Projeto

```
projeto_destaque/
├── index.html
├── style.css
├── script.js
└── logo.png
```

## 🚀 Como Usar

1. Extraia o `.zip`
2. Abra `index.html` no navegador
3. Visualize o painel e os agendamentos simulados
4. Personalize com seus próprios dados ou integre com API

## 📈 Expansões Futuras

- Integração com Firebase, Supabase ou backend Node.js
- SPA com React ou Vue.js
- Cadastro real de clientes e horários
- Painel administrativo com autenticação
